SideSync is a full-stack web application for image verification and analysis using state-of-the-art computer vision models from Hugging Face.
The project demonstrates end-to-end integration between a modern web frontend and a Python-based backend that communicates with external AI services.

What the Application Does

Allows users to upload an image through a web interface
Sends the image to a Python backend
The backend communicates with a Hugging Face CLIP BLIP models
Returns an AI-based verification / analysis result to the user

Displays the result clearly in the browser
